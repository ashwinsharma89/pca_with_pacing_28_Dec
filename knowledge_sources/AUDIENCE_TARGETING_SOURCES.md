## Audience Targeting Types

### 1. Demographic Targeting
**Attributes:**
- Age
- Gender
- Income level
- Education
- Occupation
- Marital status
- Household size

**Best For:**
- Broad audience definition
- Initial segmentation
- Combined with other targeting

### 2. Geographic Targeting
**Attributes:**
- Country
- Region/State
- City
- ZIP/Postal code
- Radius targeting
- DMA (Designated Market Area)

**Best For:**
- Local businesses
- Regional campaigns
- Store locator campaigns
- Event promotion

### 3. Behavioral Targeting
**Attributes:**
- Purchase behavior
- Website interactions
- App usage
- Content consumption
- Device usage
- Search behavior

**Best For:**
- Retargeting
- Conversion campaigns
- Personalization

### 4. Interest-Based Targeting
**Attributes:**
- Hobbies
- Interests
- Affinities
- Life events
- In-market audiences

**Best For:**
- Prospecting
- Awareness campaigns
- Lookalike expansion

### 5. Contextual Targeting
**Attributes:**
- Content topics
- Keywords
- Website categories
- Page context

**Best For:**
- Brand safety
- Relevant placements
- Privacy-compliant targeting

### 6. Lookalike/Similar Audiences
**Attributes:**
- Based on existing customers
- Similar characteristics
- Modeled audiences

**Best For:**
- Prospecting
- Scaling campaigns
- New customer acquisition

### 7. Custom Audiences
**Attributes:**
- CRM data
- Email lists
- Website visitors
- App users
- Engagement audiences

**Best For:**
- Retargeting
- Customer retention
- Upselling/cross-selling

## Audience Targeting Optimization

### Optimization Tactics
1. **Narrow Targeting**
   - Pros: Higher relevance, better conversion
   - Cons: Limited reach, higher CPM
   - Best for: Lower funnel, high-value products

2. **Broad Targeting**
   - Pros: Larger reach, discovery
   - Cons: Lower relevance, may waste budget
   - Best for: Upper funnel, awareness

3. **Layered Targeting**
   - Combine multiple criteria
   - Balance reach and precision
   - Test different combinations

4. **Exclusion Targeting**
   - Exclude existing customers
   - Exclude converters
   - Exclude irrelevant segments

5. **Sequential Targeting**
   - Target by funnel stage
   - Retarget engaged users
   - Suppress converted users
