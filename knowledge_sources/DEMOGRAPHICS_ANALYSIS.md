## Core Demographic Dimensions

### 1. Age Demographics

**Standard Age Groups:**
- 18-24 (Gen Z)
- 25-34 (Millennials)
- 35-44 (Gen X)
- 45-54 (Gen X)
- 55-64 (Baby Boomers)
- 65+ (Silent Generation)

**Marketing Considerations:**
- **18-24**: Mobile-first, social media native, value authenticity
- **25-34**: Digital savvy, career-focused, family formation
- **35-44**: Peak earning, family priorities, time-constrained
- **45-54**: Established careers, higher income, brand loyal
- **55-64**: Pre-retirement, disposable income, traditional media
- **65+**: Retirement, health-focused, value-conscious

**Platform Skews:**
- **TikTok**: 60% under 30
- **Snapchat**: 75% under 34
- **Instagram**: Younger skew (18-34)
- **Facebook**: Aging user base (25-54)
- **LinkedIn**: Professional ages (25-54)
- **Pinterest**: 25-54 primary

**Targeting Strategies:**
- Match creative to age group
- Adjust messaging tone
- Select appropriate platforms
- Consider life stage needs
- Use age-appropriate references

### 2. Gender Demographics

**Gender Categories:**
- Male
- Female
- Non-binary/Other
- Unknown/Prefer not to say

**Marketing Considerations:**
- Avoid stereotypes
- Focus on interests, not gender
- Test gender-neutral messaging
- Consider product relevance
- Respect privacy preferences

**Platform Gender Distribution:**
- **Pinterest**: 60%+ female
- **LinkedIn**: Relatively balanced
- **Instagram**: Slight female skew
- **Facebook**: Relatively balanced
- **TikTok**: Slight female skew
- **Reddit**: Male skew

**Targeting Strategies:**
- Product-specific targeting
- Interest-based over gender-based
- Test both genders for most products
- Use inclusive creative
- Monitor performance by gender

### 3. Geographic Demographics

**Geographic Levels:**
- **Country**: National campaigns
- **Region/State**: Regional targeting
- **DMA (Designated Market Area)**: TV market areas
- **City**: Urban targeting
- **ZIP/Postal Code**: Hyper-local
- **Radius**: Distance from location

**Urban vs. Rural:**
- **Urban**: Higher density, diverse, digital-first
- **Suburban**: Family-focused, car-dependent
- **Rural**: Lower density, traditional, community-oriented

**Geographic Considerations:**
- Language preferences
- Cultural differences
- Time zones
- Local events
- Weather patterns
- Economic factors
- Competitive landscape

**Targeting Strategies:**
- Localize messaging
- Adjust for time zones
- Consider local events
- Use geo-specific offers
- Test regional variations
- Monitor by location

## Demographic Analysis Techniques

### 1. Demographic Profiling

**Process:**
1. Collect demographic data
2. Identify patterns
3. Create audience segments
4. Analyze performance by segment
5. Refine targeting

**Data Sources:**
- Platform analytics
- CRM data
- Survey data
- Third-party data
- Purchase data

**Analysis Dimensions:**
- Single demographic (age only)
- Combined demographics (age + gender)
- Behavioral + demographic
- Geographic + demographic

### 2. Demographic Segmentation

**Segmentation Approaches:**

**A. Single-Variable Segmentation**
- Age groups
- Gender
- Location
- Income level

**B. Multi-Variable Segmentation**
- Age + Gender + Location
- Income + Education + Occupation
- Life stage + Household + Income

**C. Behavioral-Demographic**
- Purchase behavior + Demographics
- Engagement + Demographics
- Loyalty + Demographics

**Best Practices:**
- Start simple, add complexity
- Ensure segment size viability
- Test segment performance
- Refine based on results
- Document segment definitions

### 3. Demographic Performance Analysis

**Key Questions:**
- Which demographics convert best?
- Which have lowest CPA?
- Which have highest ROAS?
- Which engage most?
- Which have highest LTV?

**Analysis Framework:**
```
Demographic → Traffic → Engagement → Conversion → Revenue → LTV
```

**Metrics by Demographic:**
- Impressions and reach
- Click-through rate (CTR)
- Engagement rate
- Conversion rate
- Average order value (AOV)
- Cost per acquisition (CPA)
- Return on ad spend (ROAS)
- Customer lifetime value (LTV)

### 4. Demographic Comparison

**Comparison Types:**
- Demographic vs. demographic
- Demographic vs. overall average
- Demographic vs. benchmark
- Demographic over time

**Visualization Methods:**
- Bar charts (comparison)
- Heat maps (multi-dimensional)
- Scatter plots (correlation)
- Funnel charts (conversion)
- Geographic maps (location)

## Demographic Optimization Strategies

### 1. Bid Adjustments

**Strategy:**
- Increase bids for high-performing demographics
- Decrease bids for low-performing demographics
- Test incremental changes
- Monitor impact

**Recommended Adjustments:**
- +20% to +50% for top performers
- -20% to -50% for poor performers
- Start conservative, adjust based on data

### 2. Budget Allocation

**Strategy:**
- Allocate more budget to high-ROAS demographics
- Reduce budget for low-ROAS demographics
- Maintain testing budget for new demographics

**Allocation Framework:**
- 60-70% to proven demographics
- 20-30% to growth demographics
- 10% to testing new demographics

### 3. Creative Customization

**Strategy:**
- Create demographic-specific creative
- Tailor messaging to demographics
- Use appropriate imagery
- Test variations

**Customization Elements:**
- Imagery (age-appropriate, diverse)
- Messaging (tone, language)
- Offers (relevant to demographic)
- CTAs (demographic-specific)

### 4. Audience Expansion

**Strategy:**
- Identify adjacent demographics
- Test lookalike audiences
- Expand successful segments
- Monitor quality

**Expansion Tactics:**
- Age range expansion
- Geographic expansion
- Income level expansion
