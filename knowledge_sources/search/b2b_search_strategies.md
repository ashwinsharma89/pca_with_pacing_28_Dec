# B2B Search Campaign Strategies

## Overview
B2B search campaigns require fundamentally different strategies than B2C due to longer sales cycles, multiple decision-makers, higher deal values, and complex buying processes.

## Key Differences: B2B vs B2C

| Factor | B2C | B2B |
|--------|-----|-----|
| Sales Cycle | Hours to days | Weeks to months |
| Decision Makers | 1-2 | 3-10+ |
| Average Deal Value | $50-$500 | $5,000-$500,000+ |
| Purchase Frequency | High | Low |
| Emotional vs Rational | 70% emotional | 80% rational |
| Content Needs | Product info | Whitepapers, case studies, ROI calculators |

## B2B Search Strategy Framework

### 1. Keyword Strategy

#### Intent-Based Segmentation

**Problem Aware (Top of Funnel)**
- Keywords: "how to [solve problem]", "challenges with [process]"
- Example: "how to improve sales productivity"
- Bid Strategy: Lower bids, focus on impressions
- Landing Page: Educational content, blog posts
- Expected CVR: 0.5-1%

**Solution Aware (Middle of Funnel)**
- Keywords: "[solution type] software", "best [tool category]"
- Example: "CRM software for small business"
- Bid Strategy: Moderate bids
- Landing Page: Product comparison, feature pages
- Expected CVR: 2-4%

**Product Aware (Bottom of Funnel)**
- Keywords: "[your product] pricing", "[competitor] alternative"
- Example: "Salesforce alternative for startups"
- Bid Strategy: High bids, aggressive
- Landing Page: Pricing, demo request, free trial
- Expected CVR: 8-15%

**Brand Terms**
- Keywords: "[your brand]", "[your product name]"
- Bid Strategy: Maximum bids, protect brand
- Landing Page: Homepage or specific product page
- Expected CVR: 20-40%

#### B2B-Specific Keyword Patterns

**Job Title + Need**
```
"CFO financial reporting software"
"VP Sales lead generation tools"
"IT Manager security solutions"
```

**Company Size Modifiers**
```
"enterprise [solution]"
"small business [tool]"
"startup [software]"
"SMB [platform]"
```

**Industry-Specific**
```
"healthcare [solution]"
"manufacturing [software]"
"financial services [tool]"
```

**Feature-Specific**
```
"[solution] with API integration"
"[tool] with SSO"
"[platform] with custom reporting"
```

**Buying Committee Keywords**
```
"[solution] for teams"
"[tool] comparison"
"[software] ROI calculator"
"[platform] implementation cost"
```

### 3. Bidding Strategies

#### By Funnel Stage

**Top of Funnel**
- Strategy: Maximize Clicks or Manual CPC
- Target CPA: Not applicable (focus on engagement)
- Bid Adjustments: -30% to -50% from bottom-funnel
- Goal: Build awareness, gather data

**Middle of Funnel**
- Strategy: Target CPA or Maximize Conversions
- Target CPA: 2-3x your actual target
- Bid Adjustments: Baseline
- Goal: Generate qualified leads

**Bottom of Funnel**
- Strategy: Target CPA or Target ROAS
- Target CPA: Your actual target or lower
- Bid Adjustments: +20% to +50% above baseline
- Goal: Drive high-quality conversions

**Brand Terms**
- Strategy: Target Impression Share (90%+)
- Max CPC: Set ceiling to prevent overspending
- Goal: Protect brand, capture high-intent traffic

#### Bid Adjustments

**Device**
- Desktop: +20% to +40% (B2B users research at work)
- Mobile: -20% to -40% (lower conversion rates)
- Tablet: -30% to -50%

**Time of Day**
- Business Hours (9am-5pm): +10% to +30%
- Early Morning (6am-9am): +5% to +15%
- Evening (5pm-8pm): Baseline
- Night (8pm-6am): -50% to -70%

**Day of Week**
- Monday-Thursday: Baseline to +10%
- Friday: -10% to -20%
- Weekend: -40% to -70%

**Location**
- Major Business Hubs: +20% to +40%
- Target Industries Concentrated: +30% to +50%
- Low-value Regions: -30% to -50%

**Audience**
- In-Market (B2B Software): +30% to +50%
- Website Visitors (30 days): +50% to +100%
- Past Converters: +100% to +200%
- Similar Audiences: +10% to +20%

### 4. Ad Copy Best Practices

#### Headline Formulas

**Problem-Solution**
```
Headline 1: Struggling with [Problem]?
Headline 2: [Your Solution] Solves It
Headline 3: Trusted by [X] Companies
```

**Social Proof**
```
Headline 1: #1 [Solution] for [Industry]
Headline 2: Used by [Notable Clients]
Headline 3: [X]% Increase in [Metric]
```

**Feature-Benefit**
```
Headline 1: [Key Feature] That Drives Results
Headline 2: [Benefit] for [Target Audience]
Headline 3: See Why [X] Companies Choose Us
```

**Urgency/Scarcity**
```
Headline 1: Limited Time: [Offer]
Headline 2: Book Your Demo Today
Headline 3: Only [X] Spots Left This Month
```

#### Description Best Practices

**Line 1**: Value proposition + key differentiator
```
"All-in-one CRM built for scaling B2B teams. No setup fees."
```

**Line 2**: Social proof + CTA
```
"Join 5,000+ companies. Start your free 14-day trial today."
```

#### B2B-Specific Elements

**Include**:
- Industry certifications (SOC 2, GDPR compliant)
- Integration capabilities
- Customer count or notable clients
- Free trial or demo offers
- Implementation timeline
- ROI statistics

**Avoid**:
- Overly casual language
- Consumer-focused messaging
- Vague claims without proof
- Pushy sales language

### 5. Landing Page Strategy

#### Page Types by Funnel Stage

**Top of Funnel**
- **Type**: Blog post, guide, whitepaper
- **CTA**: Download resource, subscribe
- **Form Fields**: 2-3 (email, company)
- **Goal**: Capture contact, nurture

**Middle of Funnel**
- **Type**: Product page, comparison page
- **CTA**: Request demo, watch video
- **Form Fields**: 4-6 (name, email, company, role)
- **Goal**: Qualify lead, schedule demo

**Bottom of Funnel**
- **Type**: Pricing page, demo request
- **CTA**: Start trial, talk to sales
- **Form Fields**: 6-8 (full qualification)
- **Goal**: Convert to SQL

#### Landing Page Elements

**Above the Fold**:
- Clear headline matching ad promise
- Subheadline with key benefit
- Hero image/video
- Primary CTA button
- Trust indicators (logos, certifications)

**Body**:
- 3-5 key benefits with icons
- Social proof (testimonials, case studies)
- Feature highlights
- Comparison table (if relevant)
- FAQ section

**Below the Fold**:
- Secondary CTA
- Additional trust signals
- Integration logos
- Security/compliance badges

### 6. Conversion Tracking

#### Micro-Conversions
- Form submissions
- Demo requests
- Content downloads
- Video views (>50%)
- Pricing page views
- Chat initiations

#### Macro-Conversions
- Sales Qualified Leads (SQL)
- Opportunities created
- Closed-won deals
- Revenue generated

#### Tracking Setup

**Google Ads Conversions**:
1. Form submission (primary)
2. Phone call (>60 seconds)
3. Demo scheduled
4. Free trial started
5. Pricing page view (secondary)

**Offline Conversion Import**:
- SQL status
- Opportunity value
- Closed-won revenue
- Customer lifetime value

### 7. Audience Targeting

#### RLSA (Remarketing Lists for Search Ads)

**Website Visitors**
- All visitors (30 days): +30% bid
- Pricing page visitors (90 days): +100% bid
- Demo attendees (180 days): +150% bid
- Trial users (365 days): +200% bid

**Engagement-Based**
- Video viewers (>75%): +50% bid
- Content downloaders: +40% bid
- Multiple page visitors: +60% bid

**Exclusions**
- Current customers
- Employees (by IP or email domain)
- Competitors (by IP range)

#### Customer Match

**Upload Lists**:
- Past customers (for upsell/cross-sell)
- SQL list (for nurture)
- Event attendees
- Webinar registrants

**Bid Adjustments**:
- Past customers: +100% to +200%
- SQLs: +80% to +150%
- MQLs: +50% to +100%

### 8. Budget Allocation

#### By Funnel Stage

**Recommended Split**:
- Brand: 20-30%
- Bottom Funnel: 40-50%
- Middle Funnel: 20-30%
- Top Funnel: 5-10%
- Competitor: 5-10%

#### By Campaign Performance

**High Performers** (ROAS > 5:1):
- Allocate 60-70% of budget
- Maximize impression share
- Test expansion opportunities

**Medium Performers** (ROAS 2-5:1):
- Allocate 20-30% of budget
- Optimize for efficiency
- Test new keywords/audiences

**Low Performers** (ROAS < 2:1):
- Allocate 5-10% of budget
- Test and learn
- Consider pausing if no improvement

### 9. Account-Based Marketing (ABM) Integration

#### Target Account Lists

**Tier 1 (Top 50 accounts)**:
- Custom intent audiences
- Company name keywords
- LinkedIn + Search combo
- Maximum bids

**Tier 2 (Next 200 accounts)**:
- Industry + size targeting
- Firmographic audiences
- Elevated bids

**Tier 3 (Next 1000 accounts)**:
- Broad industry targeting
- Standard bids
- Awareness focus

#### ABM Tactics

**Search Ads**:
- Bid on target company names
- Use dynamic keyword insertion for company names
- Create custom landing pages per account

**RLSA**:
- Target visitors from target accounts
- Personalized messaging
- Higher bids for target accounts

### 10. Measurement & Optimization

#### Key Metrics

**Lead Quality Metrics**:
- MQL to SQL conversion rate: 20-30%
- SQL to Opportunity: 30-50%
- Opportunity to Close: 20-40%
- Average deal size
- Sales cycle length

**Campaign Metrics**:
- Cost per MQL: $50-$200
- Cost per SQL: $200-$1,000
- Cost per Opportunity: $500-$3,000
- Cost per Customer: $2,000-$20,000+
- Customer Acquisition Cost (CAC): Varies widely

**Efficiency Metrics**:
- CAC Payback Period: <12 months
- LTV:CAC Ratio: >3:1
- Marketing % of CAC: <30%

#### Optimization Cadence

**Daily**:
- Monitor spend pacing
- Check for disapproved ads
- Review search term reports

**Weekly**:
- Add negative keywords
- Adjust bids based on performance
- Test new ad copy

**Monthly**:
- Comprehensive performance review
- Budget reallocation
- Audience analysis
- Competitor analysis

**Quarterly**:
- Strategy review
- Landing page updates
- Attribution model review
- Sales alignment check

## Common B2B Search Mistakes

### 1. Using B2C Tactics
**Problem**: Short-term focus, wrong metrics
**Solution**: Focus on lead quality, not just volume

### 2. Ignoring Sales Feedback
**Problem**: Generating low-quality leads
**Solution**: Regular sales-marketing alignment meetings

### 3. Not Tracking Offline Conversions
**Problem**: Incomplete attribution, poor optimization
**Solution**: Import offline conversion data

### 4. Bidding Too Aggressively on Top Funnel
**Problem**: Wasted budget on unqualified traffic
**Solution**: Lower bids, focus on engagement metrics

### 5. Neglecting Negative Keywords
**Problem**: Irrelevant clicks, wasted spend
**Solution**: Weekly search term review, comprehensive negative list

## Advanced Strategies

### 1. Intent-Based Audiences
- Layer in-market audiences on search campaigns
- Create custom intent audiences from competitor research
- Use similar audiences for prospecting

### 2. Dynamic Search Ads (DSA)
- Use for long-tail keyword coverage
- Set up for specific website sections
- Lower bids than standard search

### 3. Responsive Search Ads (RSA)
- Test 10-15 headlines
- Include dynamic keyword insertion
- Pin key messages to specific positions

### 4. Call-Only Campaigns
- Target mobile users during business hours
- Use call extensions with tracking
- Optimize for call conversions

### 5. Local Search for Multi-Location
- Target specific office locations
- Use location extensions
- Customize ads by region
